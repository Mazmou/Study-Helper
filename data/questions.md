## Q4. What does the sliding window mechanism in TCP manage?

- A) Connection establishment
- B) Data ordering
- C) Error detection
- D) Flow control

**Correct:** D) Flow control

**Evidence:** TCP uses a sliding window mechanism to manage flow control. The receiver advertises a window size that determines how much data the sender may transmit before receiving an acknowledgment.
