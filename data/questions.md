## Q3. What is HTTP used for?

- A) Routing packets across networks
- B) Securing network communication
- C) Managing flow control
- D) Transferring hypermedia documents

**Correct:** D) Transferring hypermedia documents

**Evidence:** Hypertext Transfer Protocol (HTTP) is an application-layer protocol used for transferring hypermedia documents such as HTML.

## Q4. What does the sliding window mechanism in TCP manage?

- A) Packet fragmentation
- B) Data encryption
- C) Flow control
- D) Error detection

**Correct:** C) Flow control

**Evidence:** TCP uses a sliding window mechanism to manage flow control. The receiver advertises a window size that determines how much data the sender may transmit before receiving an acknowledgment.
